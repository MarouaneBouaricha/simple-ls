# Simple-ls
Implementation of ls command in the C language

## Description:
the program should show :
- file type
- file persmissions
- file size
- date of creation 
- name of the file (of course)
- hidden file using -a flag

The impemlention made with system call like : stat, dirent ..etc

## Usage:
```
> gcc lsCmd.c -o lst
> ./lst directoryname
```
